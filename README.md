# Cloud Native Order Service

Production-ready Order Management Service built using Java 21, Spring Boot and PostgreSQL.

## Tech Stack

- Java 21
- Spring Boot 3
- PostgreSQL
- Maven
- Docker (Planned)
- GitHub Actions (Planned)
- Kubernetes (Planned)
- OpenTelemetry (Planned)
- Prometheus & Grafana (Planned)

## Features

- Create Orders
- Retrieve Orders
- Update Order Status
- Delete Orders
- Input Validation
- Global Exception Handling
- Logging

## Architecture

Controller
↓
Service
↓
Repository
↓
PostgreSQL
